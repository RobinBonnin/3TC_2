Segmente un intervalle (entre les positions debut et fin) en deux sous-intervalles dont les entiers se trouvent de part et d'autre d'un entier pivot. La fonction choisit un entier (le premier rencontre) et rearrange les elements du tableau d de telle sorte que : d[debut..pivot-1] <= d[pivot] < d[pivor+1..fin].

Parameters
dun tableau de N entiers non necessairement trie
Nla taille du tableau
debutl'indice inferieur de l'intervalle a trier
finl'indice superieur de l'intervalle a trier
Precondition
0 <= debut < fin < N
Postcondition
d[debut..pivot-1] <= d[pivot] < d[pivor+1..fin]
Returns
pivot la position de l'entier pivot
